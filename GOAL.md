# Goal

Build a smoothly playable, realistic, exciting car-only open-world driving game in a Gaussian-splat city world.

The real success criterion is the player experience: the game should feel increasingly real, smooth, alive, responsive, and fun to drive.

Engineering work matters only insofar as it improves that experience or removes a blocker to improving it.

Do not optimize for tests, infrastructure, research, or agent activity as ends in themselves.

Python is only used for the autonomous orchestration system. The game target is Unreal Engine 5 with C++/Blueprints/plugins as needed.
