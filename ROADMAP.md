# Roadmap

## M0 - Agent OS Boots

The VPS daemon can run the orchestrator loop, spawn worker/reviewer tasks, update state, freeze/resume, and manage PRs.

## M1 - Research-To-Decision Loop

The system researches UE5 Gaussian-splat integration and records a justified path.

## M2 - Minimal Game Shell

A UE5 project or prototype shell exists and can be run.

## M3 - Drivable Test Loop

The player can drive a car on a simple loop with stable controls and camera.

## M4 - Splat World Prototype

A small Gaussian-splat scene renders with an aligned drivable collision surface.

## M5 - Experience Improvement Loop

The system can use playtest evidence, screenshots, video, or telemetry to choose the next improvement.

## M6 - Living Traffic Slice

Traffic cars make the scene feel more alive and interactive.

## M7 - Stable Vertical Slice

A short route is smooth, realistic, exciting, and repeatedly playable.
